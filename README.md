# Rock-Paper-Scissors

Un juego clásico de Piedra, Papel o Tijeras implementado en JavaScript para practicar conceptos básicos de programación y lógica.

## Descripción

Este proyecto es una implementación del juego tradicional de Piedra, Papel o Tijeras que se ejecuta en la consola del navegador. Fue creado como práctica para reforzar conceptos fundamentales de JavaScript.

## Cómo Jugar

1. Abre el archivo `index.html` en tu navegador
2. Abre la consola del navegador (F12 o clic derecho → Inspeccionar → Consola)
3. El juego se ejecutará automáticamente mostrando los resultados en la consola

## Tecnologías Utilizadas

- HTML5
- CSS
- JavaScript
